les bases du HTML

HTML est un langage de balisage utilisé pour créer des pages web. Il est basé sur le concept de balises qui permettent de structurer le contenu d'une page web.

les bases du CSS

CSS est un langage de style utilisé pour styliser les pages web. Il est basé sur le concept de sélecteurs qui permettent de sélectionner les éléments d'une page web et de les styliser.

les bases du JavaScript

JavaScript est un langage de programmation utilisé pour créer des pages web interactives. Il est basé sur le concept de fonctions qui permettent de créer des fonctionnalités sur une page web.

les bases du PHP

PHP est un langage de programmation utilisé pour créer des pages web dynamiques. Il est basé sur le concept de variables qui permettent de stocker des données sur une page web.

les bases du MySQL

MySQL est un système de gestion de base de données utilisé pour stocker des données sur une page web.
